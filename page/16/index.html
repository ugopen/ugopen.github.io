NX二次开发-获取两条曲线的交点（UF_CURVE_intersect ）

发表于 2020-12-22 分类于 NX二次开发

Overview Determine intersection between curve/edge and a curve/edge/face/single face body/plane/datum plane. The intersection will be done in the orientation of the current WCS when the entity is a curve/edge otherwise in 3D. If the intersection is an actual 3D intersectionthen that information will be output. Also the intersection between the naturallyextended version of the curve or edge will be returned if needed, however, only a curve(the first calling parameter) will be extended when the entity(the second calling parameter) is something other than a curve/edge.

For curve (the first calling parameter) of UF_line_type or UF_spline_type, a return parameter greater than 0 or less than 1, indicates that the itersection is on the natural extension of the curve.

确定曲线/边与曲线/边/面/单面体/平面/基准面的交点。当实体是曲线/边时，交叉将在当前WCS的方向上完成，否则在3D中。如果交点是实数 然后这些信息就会被输出。同样，如果需要，自然扩展的曲线或边之间的交集也将被返回，但是，当实体(第二个调用参数)不是曲线/边时，只会扩展一条曲线(第一个调用参数)。
