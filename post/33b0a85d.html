    网络连接概念

    IP地址IPADDR

    • IP地址是一种逻辑地址，用来标识网络中一个个主机
    1. IP地址=网络地址+主机地址
    2. IP地址是一个 4 * 8bit（1字节）由 0/1 组成的数字串（IP4协议）
    • 子网掩码NETMASK
    1. 子网掩码只有一个功能，就是将IP地址划分为网络地址和主机地址两部分
    2. 子网掩码用来判断任意两台计算机的IP地址是否在同一个子网中的根据
    3. （A 192.168.7.111， B 192.168.8.222）
    4. （255.255.0.0）
    • 默认网关GATEWAY
    1. 连接两个不同的网络的设备都可以叫网关设备；网关的作用就是实现两个网络之间进行通讯与控制
    2. 网关地址就是网关设备的IP地址
    • 域名服务器DNS
    1. DNS是域名服务器，用来解析域名的（域名和IP之间的解析）
    2. 如果没有这东西，登陆某个网站时就必须输入该网站的IP地址，有了DNS就可以直接输入网址
    3. C:\Windows\System32\drivers\etc\hosts（本地主机配置文件：127.0.0.1）

    网络连接模式

    host-onboy(主机模式)

    • 在某些特殊的网络调试环境中，要求将真实环境和虚拟环境隔离开，这时你就可采用hostonboy模式
    • 在host-onboy模式中，所有的虚拟系统是可以相互通信的，但虚拟系统和真实的网络是被隔离开的
    • 在host-onboy模式下，虚拟系统的TCP/IP配置信息都是由VMnet1(host-onboy)虚拟网络的DHCP服务器来动态分配的

    bridged(桥接模式)

    • VMWare虚拟出来的操作系统就像是局域网中的一台独立的主机，它可以访问网内任何一台机 器
    • 使用桥接模式的虚拟系统和宿主机器的关系，就像连接在同一个Hub上的两台电脑
    • 当前主机IP 为 192.168.8.100 虚拟机 192.168.8.xxx（学习期间为了防止IP冲突，所以不选择这种模式）

    NAT(网络地址转换模式)

    • 使用NAT模式，就是让虚拟系统借助NAT(网络地址转换)功能，通过宿主机器所在的网络来访问公网
    • NAT模式下的虚拟系统的TCP/IP配置信息是由VMnet8(NAT)虚拟网络的DHCP服务器提供的虚拟系统也就无法和本局域网中的其他真实主机进行通讯

    软件分类

    应用软件

    • 就是为了实现某些业务功能
    • 应用软件要基于对应的系统软件
    • 不同的操作系统要安装不同的软件

    系统软件

    • 就是为了和硬件打交道
    • 屏蔽应用软件与硬件的差异

    系统软件的分类

    • Window
    1. 用户量全球最大
    2. 收费，不开源，民用较多
    3. 各种软件比较齐全
    • Mac
    1. 只限定于某些苹果的品牌机
    2. ios–自成一家
    • GNU/Linux
    1. GNU是一个开源软件组织,世界上所有的软件都应该开源免费
    2. GNU Is Not Unix
    3. GCC++
    4. 托瓦兹 林纳斯 Linus – Linux(Linux is not unix)
    5. Logo是企鹅

    Linux分支

    RedHat（收费）

    • CentOS
    1. 完全开源免费
    2. 不要使用最新版的CentOS
    • 主要用于服务器版本

    Debain（免费）

    • Ubuntu
    1. 视窗界面良好的Linux系统
    2. 一些主流的软件都支持Ubuntu系统

    镜像的分类

    This image contains the complete set of packages for CentOS Linux 7. It can be used for installing or populating a local mirror.

    此映像包含用于CentOS Linux 7的完整软件包集。它可以用于安装或填充本地镜像。

    The aim of this image is to install a very basic CentOS 6.5 system with the minimum of packages needed to have a functional system.

    这个映像的目的是安装一个非常基本的CentOS 6.5系统，使用拥有一个功能系统所需的最小包。

    This is the network install and rescue image.

    这是网络安装和救援映像。
