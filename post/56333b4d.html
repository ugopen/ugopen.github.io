    Linux 基础教程六：VI 和 VIM 编辑器

    VI和VIM编辑器

    打开文件

    三种模式

    • 编辑模式-–编辑模式中，每一个按键都有其他的功能
    • 输入模式-–每一个按键按下什么，就像文本中数据输入什么
    • 末行（命令行）模式-–我们可以直接在VI中输入特定的命令

    三种模式切换

    光标操作（命令模式）

    命令 描述
    j 下移
    k 上移
    h 左移
    l 右移
    0（数字） 将光标移动到行首
    $ 将光标移动到行尾
    b 将光标移动到单词首字符
    e 将光标移动到单词未字符
    % 括号对应查找
    ( 将光标移动到段落首行
    ) 将光标移动到段落未尾
    w 将光标移动到下个单词行首
    gg 将光标移动到文件首行
    G 将光标移动到文件未行
    H 将光标移动到页首
    M 将光标移动到页中
    L 将光标移动到页尾

    进入编辑模式

    命令 描述
    i 在当前字符前开始编辑
    I 在当前行首开始编辑
    a 在当前字符后开始编辑
    A 在当前行尾开始编辑
    o 在当前字符下方插入行开始编辑
    O 在当前字符上方插入行开始编辑
    s 删除光标所选字符，开始编辑
    S 删除光标所在行，开始编辑

    辅助操作（命令模式）

    命令 描述
    ctrl + shift +t 终端左右分屏
    gg=G 自动格式化程序
    shuft + k 查看帮助文档
    [d 查看定义语句，将光标移动到待查看函数单词上
    ~ 转换大小写
    u 撤销
    ctrl + r 反向撤销
    dd 删除整行
    dw 删除单词（前提条件光标需要在单词首字符位置处）
    yy 复制整行
    x（小写） 剪切（不带回车）
    X（大写） 剪切（带回车）
    p（小写） 粘贴（在光标位置处，下行粘贴，提前带回车符号）或者（在光标后面粘贴）
    P（大写） 粘贴（在光标位置处，上行粘贴，提前带回车符号）或者（在光标前面粘贴）
    / （比如：”/emmet”） 查找命令，通过“n”继续查找下个字符或单词
    * 查找命令，在单词位置处，通过“*”命令查找下一个此单词位置
    # 查找命令，在单词位置处，通过“#”命令查找上一个此单词位置
    ctrl + ww 切换窗口 （分屏后操作）
    ctrl “+” w “+” = 所有窗口设置成适合大小（分屏后操作）

    末行模式

    命令 描述
    :w 保存文件
    :q 退出
    :vsp 左右（分屏操作）
    :sp 上下（分屏操作）
    :s/原数据/新数据 单行替换，将光标移动到替换行上
    :%s/原数据/新数据 单行首个替换，将光标移动到替换行上
    :%s/原数据/新数据/g 全文替换，将光标移动到替换行上
    :起始行号,终止行号%s/原数据/新数据/g 指定区域全文替换，将光标移动到替换行上
