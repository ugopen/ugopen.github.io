    教程目录《C++基础教程》

    第八章：结构体

    8.6 结构体做函数参数

    作用：将结构体作为参数向函数中传递

    传递方式有两种：

    • 值传递
    • 地址传递

    示例：

    #include <iostream>
using namespace std;
#include <string>

//定义结构体
struct GroupLeader
{
//成员列表
int iNo;//工号
string sName;//姓名
int iAge;//年龄
};

//值传递
void PrintGroupLeader01(GroupLeader s1)
{
s1.sName = "黄民";
//打印显示
cout << "01子函数\\n" << "组长工号：" << s1.iNo << " 组长姓名：" << s1.sName << " 组长年龄：" << s1.iAge << endl << endl;
} 

//地址传递 
void PrintGroupLeader02(GroupLeader \*p)
{ p->sName = "李四";
//打印显示
cout << "02子函数\\n" << "组长工号：" << p->iNo << " 组长姓名：" << p->sName << " 组长年龄：" << p->iAge << endl << endl;
}

int main()
{
//定义组长参数列表
GroupLeader s1;
s1.iNo = 0001;
s1.sName = "黄家驹";
s1.iAge = 60;

PrintGroupLeader01(s1);//值传递
PrintGroupLeader02(&s1);//地址传递

//打印显示
cout << "main\\n" << "组长工号：" << s1.iNo << " 组长姓名：" << s1.sName << " 组长年龄：" << s1.iAge << endl;

system("pause");

return 0;
}

    总结：如果不想修改主函数中的数据用值传递，反之使用地址传递。
