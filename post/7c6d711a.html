    Linux 基础教程四：Linux 命令

    发表于 2022-03-30 分类于 Linux新手村

    命令学习法

    Linux命令与参数之间必须用空格隔开

    Linux命令是区分大小写的

    如果输入了错误的命令

    • -bash: abcd: command not found
    • 命令敲错了
    • 命令未安装

    type 命令的类型

    • cd is a shell builtin
    • ping is /bin/ping
    • ll is aliased to `ls -l –color=auto’
    • for is a shell keyword

    命令的帮助文档

    **help—**内置命令的帮助文档

    man-–外部命令的帮助文档

    1. 因为当前系统为minimal的安装包，very basic 没有man包
    2. 需要手动安装man（yum install man man-pages -y）

    将来工作中如果遇到生疏的命令，直接百度

    1. 如果不是为了装C,完全没必要查看命令手册

    常用的命令

    命令 描述
    whereis（例如：whereis cd） 查询命令文件的位置
    file（例如：file anaconda-ks.cfg） 查看文件的类型
    who 查看当前在线的用户
    whoami 我是谁
    pwd 我在那
    uname -a 查看内核信息
    echo 类似于 sout syso 打印语句
    clear 清屏
    history 历史
